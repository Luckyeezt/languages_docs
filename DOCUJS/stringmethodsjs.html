Méthodes de Manipulation de Chaînes

Voici les méthodes les plus couramment utilisées pour manipuler des chaînes de caractères en JavaScript :

  • length : Retourne la longueur d'une chaîne de caractères.
  • charAt() : Retourne le caractère à l'index spécifié.
  • indexOf() : Retourne l'index de la première occurrence de la chaîne spécifiée.
  • lastIndexOf() : Retourne l'index de la dernière occurrence de la chaîne spécifiée.
  • toLowerCase() : Convertit une chaîne de caractères en minuscules.
  • toUpperCase() : Convertit une chaîne de caractères en majuscules.
  • trim() : Supprime les espaces blancs au début et à la fin d'une chaîne de caractères.

  • includes() :

  • replace() : Remplace une partie d'une chaîne par une autre.

La propriété .length

La propriété .length est une propriété des chaînes de caractères en JavaScript qui renvoie la longueur de la chaîne. Elle compte le nombre de caractères dans la chaîne, y compris les espaces et les caractères spéciaux.



Exemple d'utilisation de .length

console.log("Bonjour".length); // Console : 7
console.log("Salut tout le monde".length); // Console : 19
console.log("Bonsoir DOCUMENTATion".length); // Console : 21


Syntaxe :

  • "abc".length



La propriété .length est souvent utilisée pour :


  • Valider la longueur d'une chaîne avant de la traiter.
  • Boucler à travers chaque caractère d'une chaîne.
  • Déterminer la longueur d'un tableau (pour les tableaux).
  • Effectuer des opérations basées sur la longueur de la chaîne.
  • La propriété .length compte même les espaces en tant que caractère !


  • Traduction : length veut dire "longueur" en français => la longueur du string.

    • La méthode .charAt()

    La méthode .charAt() est une méthode des chaînes de caractères en JavaScript qui renvoie le caractère à la position spécifiée dans la chaîne. Les positions commencent à partir de zéro (0) pour le premier caractère.



    Exemple d'utilisation de .charAt()

    console.log("Bonjour".charAt(0)); // Console : b
console.log("Documentation Javascript".charAt(7)); // Console : t
console.log(" Youtube".charAt(0)); // Console :  (vide)
console.log("@123@clash54".charAt(4)); // Console : @


    Syntaxe :

    • "abc".charAt(nombre)



    La méthode .charAt() est souvent utilisée pour :


    • Accéder à un caractère spécifique dans une chaîne de caractères.
    • Itérer à travers chaque caractère d'une chaîne.
    • Vérifier ou manipuler des caractères individuels dans une chaîne.


  • Traduction : charAt => CHARacter AT => La place du caractère.

    • La méthode .indexOf()

    La méthode .indexOf() est une méthode des chaînes de caractères en JavaScript qui renvoie l'index de la première occurrence d'un caractère ou d'une sous-chaîne dans une chaîne. Si la sous-chaîne n'est pas trouvée, la méthode renvoie -1.



    Exemple d'utilisation de .indexOf()

    console.log("Bonjour".indexOf('o')); // Console : 1
console.log("Documentation Javascript".indexOf('script')); // Console : 13
console.log("Youtube".indexOf('T')); // Console : -1 (car la casse est différente)
console.log("JavaScript".indexOf('Java')); // Console : 0


    Syntaxe :

    • "abc".indexOf('caractère')



    La méthode .indexOf() est souvent utilisée pour :


    • Rechercher la première occurrence d'un caractère ou d'une sous-chaîne dans une chaîne.
    • Déterminer si une chaîne contient une sous-chaîne spécifique.
    • Trouver la position d'un caractère ou d'une sous-chaîne dans une chaîne.


  • Traduction : indexOf => Index Of => Indice de la position.

    • La méthode .lastIndexOf()

    La méthode .lastIndexOf() est une méthode des chaînes de caractères en JavaScript qui renvoie l'index de la dernière occurrence d'un caractère ou d'une sous-chaîne dans une chaîne. La recherche commence à partir de la fin de la chaîne. Si la sous-chaîne n'est pas trouvée, la méthode renvoie -1.



    Exemple d'utilisation de .lastIndexOf()

    console.log("Bonjour".lastIndexOf('o')); // Console : 3
console.log("Documentation Javascript".lastIndexOf('script')); // Console : 14
console.log("Youtube".lastIndexOf('T')); // Console : -1 (car la casse est différente)
console.log("JavaScript".lastIndexOf('a')); // Console : 3


    Syntaxe :

    • "abc".lastIndexOf('caractère')



    La méthode .lastIndexOf() est souvent utilisée pour :


    • Rechercher la dernière occurrence d'un caractère ou d'une sous-chaîne dans une chaîne.
    • Déterminer si une chaîne contient une sous-chaîne spécifique à partir de la fin.
    • Trouver la position de la dernière occurrence d'un caractère ou d'une sous-chaîne dans une chaîne.


  • Traduction : lastIndexOf => Last Index Of => Dernier indice de la position.

console.log('HELLO'.toLowerCase()); // Console : hello

  • Traduction : toLowerCase => To Lower Case => En minuscules.

    • La méthode .trim()

    La méthode .trim() est une méthode des chaînes de caractères en JavaScript qui supprime les espaces blancs inutiles au début et à la fin d'une chaîne.



    Exemple d'utilisation de .trim()

    console.log('    Bonjour    '.trim()); // Console : 'Bonjour'
console.log('   HELLO   '.trim()); // Console : 'HELLO'
console.log('   JavaScript   '.trim()); // Console : 'JavaScript'


    Syntaxe :

    • " abc ".trim()



    La méthode .trim() est souvent utilisée pour :


    • Supprimer les espaces inutiles autour des entrées de l'utilisateur dans les formulaires web.
    • Nettoyer les données importées ou récupérées pour éviter les erreurs dues aux espaces inutiles.
    • Faciliter le traitement et la comparaison des chaînes de caractères.


  • Traduction : trim => Trim => Couper.

    • La méthode .repeat()

    La méthode .repeat() est une méthode des chaînes de caractères en JavaScript qui renvoie une nouvelle chaîne composée du nombre de copies spécifié de la chaîne d'origine concaténées ensemble.



    Exemple d'utilisation de .repeat()

    console.log('A'.repeat(5)); // Console : 'AAAAA'
console.log('Hello '.repeat(3)); // Console : 'Hello Hello Hello '
console.log('123'.repeat(2)); // Console : '123123'


    Syntaxe :

    • "abc".repeat(nombre de répétitions)



    La méthode .repeat() est souvent utilisée pour :


    • Créer une chaîne répétée un certain nombre de fois.
    • Construire des chaînes de caractères pour les tests et la génération de données.
    • Formater des chaînes de caractères avec des motifs répétés.


  • Traduction : repeat => Repeat => Répéter.

    • La méthode .substring()

    La méthode .substring() est une méthode des chaînes de caractères en JavaScript qui extrait une sous-chaîne de caractères d'une chaîne d'origine, en fonction des indices de début et de fin spécifiés.



    Exemple d'utilisation de .substring()

    console.log('abcdef'.substring(2, 4)); // Console : 'cd'
console.log('Hello, world!'.substring(7)); // Console : 'world!'
console.log('JavaScript'.substring(4)); // Console : 'Script'


    Syntaxe :

    • "abc".substring(2, 4)



    La méthode .substring() est souvent utilisée pour :


    • Extraire des sous-chaînes de caractères d'une chaîne d'origine pour la manipulation et l'affichage.
    • Tronquer ou découper une partie d'une chaîne en fonction des besoins de traitement.
    • Manipuler des chaînes de caractères pour obtenir des portions spécifiques de texte.


  • Traduction : substring => Substring => Sous-chaîne.

    • La méthode .replace()

    La méthode .replace() est une méthode des chaînes de caractères en JavaScript qui remplace une partie d'une chaîne par une autre chaîne ou une valeur spécifiée.



    Exemple d'utilisation de .replace()

    console.log('Hello, world!'.replace(/world/, 'friend')); // Console : 'Hello, friend!'
console.log('I like apples and oranges.'.replace(/apples and oranges/, 'bananas')); // Console : 'I like bananas.'
console.log('Hello, world!'.replace(/hello/i, 'Hi')); // Console : 'Hi, world!'


    Syntaxe :

    • "abc".replace("a", "b")



    La méthode .replace() est souvent utilisée pour :


    • Modifier des portions de texte dans une chaîne selon des critères spécifiques.
    • Effectuer des remplacements de chaînes de manière conditionnelle.
    • Effectuer des substitutions de texte pour le traitement des données.


  • Traduction : replace => Replace => Remplacer.

    • La fonction Math.sin()

    La fonction Math.sin() est une méthode de l'objet Math en JavaScript qui permet de calculer le sinus d'un nombre. Elle prend un seul argument, qui est l'angle en radians dont vous souhaitez calculer le sinus, et retourne le sinus de cet angle.


    • Rappel Mathématique : Le sinus (sin) en mathématiques est une fonction trigonométrique qui associe à un angle dans un triangle rectangle le rapport entre la longueur du côté opposé à cet angle et l'hypoténuse du triangle.


    Exemples de l'utilisation de Math.sin() sur des nombres : 

    // Exemple de l'utilisation de Math.sin() sur des nombres positifs :
console.log(Math.sin(90)); // Console : 0.8939966636005579
console.log(Math.sin(180)); // Console : -0.8011526357338304

// Exemple de l'utilisation de Math.sin() sur un nombre négatif :
console.log(Math.sin(-360)); // Console : -0.9589157234143065


    • L'argument passé à Math.sin() est l'angle en radians pour lequel vous souhaitez calculer le sinus
    • Le sinus d'un angle dans un triangle rectangle est défini comme le rapport de la longueur du côté opposé à l'angle sur l'hypoténuse du triangle.
    • Trad : "sin" est l'abbréviation de "sine" qui veut dire "sinus" en français.

console.log(Math.cos(180)); // Console : -0.5984600690578581

// Exemple de l'utilisation de Math.cos() sur un nombre négatif :
console.log(Math.cos(-360)); // Console : -0.2836910914865273


    • L'argument passé à Math.cos() est l'angle en radians pour lequel vous souhaitez calculer le cosinus.
    • Le cosinus d'un angle dans un triangle rectangle est défini comme le rapport de la longueur du côté adjacent à l'angle sur l'hypoténuse du triangle.
    • Trad : "cos" est l'abbréviation de "cosine" qui veut dire "cosinus" en français.

    La fonction Math.random()

    La fonction Math.random() en JavaScript est une fonction intégrée qui génère un nombre aléatoire compris dans l'intervalle [0, 1). Cela signifie que Math.random() retourne un nombre décimal pseudo-aléatoire qui peut être égal ou supérieur à 0 (inclus) et strictement inférieur à 1 (exclus).



    Exemple qui génère un nombre aléatoire entre 0 et 10 :

    L'utilisation de la fonction Math.floor permet de retirer la partie décimale en retournant le plus grand entier inférieur et génère ainsi un nombre aléatoire entier au lieu d'un nombre décimal.

    let num = 10; // Création de la variable "num" qui stocke le nombre "10".
const gen = Math.floor(Math.random() * number); // Création de la variable "gen" qui emploie la fonction Math.random() sur la variable "num".
console.log(gen); // Console : Un nombre aléatoire entre 0 et 10.


    Exemple qui génère un nombre entre 1 et 100 :

    let randomNum = Math.floor(Math.random() * 100 + 1); // La plage de nombres aléatoires sera de 1 (inclus) à 101 (exclus), couvrant ainsi les nombres entiers de 1 à 100.
console.log(randomNum); // Console : Un nombre aléatoire entre 1 et 100.
    • La fonction Math.random() ne prend aucun argument.
    • Trad : "random" veut dire "aléatoire" en français.

    La fonction Math.min()

    La fonction Math.min() est une méthode de l'objet Math en JavaScript qui permet de trouver le plus petit nombre parmi un ensemble d'arguments ou d'éléments. Elle peut être utilisée pour retourner le nombre le plus petit dans une liste d'arguments ou dans un tableau.



    Exemples qui cherche et affiche le nombre minimum : 

    console.log(Math.min(5, 10, 2, 100)) // Console : 2


    • La fonction Math.min() peut prendre autant d'arguments numériques qu'elle veut.
    • Elle retourne le plus petit nombre parmi les valeurs fournies en argument.
    • Si vous fournissez à la fonction Math.min() des valeurs non numériques, elle tentera de les convertir en nombres avant de les comparer.
    • Trad : "min" est l'abbréviation de "minimum" en français.

    Exemples qui cherche et affiche le nombre minimum : 

    console.log(Math.max(5, 10, 2, 100)) // Console : 100


    • La fonction Math.max() peut prendre autant d'arguments numériques qu'elle veut.
    • Elle retourne le plus grand nombre parmi les valeurs fournies en argument.
    • Si vous fournissez à la fonction Math.max() des valeurs non numériques, elle tentera de les convertir en nombres avant de les comparer.
    • Trad : "max" est l'abbréviation de "maximmum" en français.
