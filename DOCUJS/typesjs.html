Les différents types de données en Javascript

  • String : C'est une séquence de caractères comme des lettres, chiffres, symboles pour représenter un texte en Javascript.
  • Boolean : Permet de représenter une valeur logique avec les booléens true(vrai) et false(faux).
  • Number : Permet de réprésenter des valeurs numériques comme un nombre entier ou un nombre décimal.
  • undefined : Valeur par défaut si une variable à été déclarée mais aucune valeur lui à été assignée.
  • null : Réprésente l'absence volontaire de valeur pour une variable.
  • Array : Permet de stocker une collection ordonnée d'élements, offre une manipulation de données efficace telle qu'une liste.
  • Object : Permet de regrouper des données et des fonctionnalités dans une seule unité.
  • L'opérateur typeof : Permet de savoir le type de données d'une valeur.

  • >>>> En Javascript, il existe deux types de données, il existe des données primitifs, comme les booléens, les chaines de caractères, les nombres etc..., c'est à dire que c'est une valeur qui ne peut pas changer (je parle pas des variables). D'un autre côté, il existe des types de données non-primitifs, qui sont les objets et les tableaux, où l'on peut changer leurs valeurs avec des méthodes

Le type : String (chaine de caractères)

Un "string" est une chaine de caractères, cela peut être un mot, une expression ou une phrase.

Comme tout élément en Javascript, le string à 3 syntaxes particulières (3 façons de s'écrire) :


  • Guillemets doubles : -> Utilisés pour créer une chaîne de caractères.
  • Guillemets simples : -> Utilisés également pour créer une chaîne de caractères.
  • Backticks : -> Utilisés pour créer une chaïne de caractères sur plusieurs lignes (évitant la concaténation) et est utilisé pour les interpolations de variables.
"Bienvenue sur notre site !" // Ici on utilise les guillemets doubles.
'Bienvenue sur notre site !' // Ici on utilise les guillemets simples.
`Bienvenue sur notre site !` // Ici on utilise les backticks.


Les séquences d'échappement sur les chaînes de caractères

  • \n -> Permet de sauter une ligne dans une chaîne de caractères.
  • \t -> Tabulation, permet de faire 8 espaces lors d'une chaîne de caractères.
  • \ -> (Barre oblique inverse) Elle permet d'indiquer au langage que le caractère qui suit doit être interprété différemment. Par exemple, \' est utilisé pour représenter une simple citation (') sans que JavaScript ne la confonde avec la fin de la chaîne de caractères.


Littéraux de modèles (Interpolation de variables) :

Afin d'implémenter des variables dans une chaine de caractères, nous devons utiliser déjà ce que l'on appelle les backticks : ` `

let name = "Nicolas";
console.log(`Le nom de l'utilisateur est : ${name}`) // Le nom de l'utilisateur est Nicolas.

Le type : Number (valeurs numériques)

Un "number" est une valeur numérique en JavaScript, pouvant être un entier ou un nombre décimal.

Les nombres s'utilisent principalement pour les opérations, en Javascript nous pouvons écrire un nombre entier et décimal qu'il soit positif ou négatif :

  2.5 // Ceci est un nombre décimal positif.
  1000000 // Ceci est un nombre entier positif.
  -50.25 // Ceci est un nombre décimal négatif.
  -100 // Ceci est un nombre entier négatif.

Le type : Boolean (instructions logiques)

Un "boolean" contient seulement deux valeurs "true" (vrai) et "false" (faux)

Les booléens s'utilisent comme des conditions vraies ou fausses, principalement utilisés pour vérifier si quelque chose est vrai ou faux (avec une comparaison par exemple), dans ce cas true reflète "oui" et false reflète "non"

  • Tout les nombres positifs ou négatifs sauf 0, Toutes les chaines de caractères sauf une chaine de caractères vide (" ") et le booléen true sont considérés comme vraies.
  • Le nombre 0, null,undefined,NaN,the boolean False et une chaine vide sont considérés comme étant faux.
  true // true signifie vrai.
  false // false signifie faux.

Le type : undefined

En JavaScript, si nous n'attribuons pas de valeur à une variable, la valeur n'est pas définie. De plus, si une fonction ne renvoie rien, elle renvoie undefined.

let youtubeur; // Nous assignons pas de valeur à la variable "youtubeur"
console.log(youtubeur) // Renvoie undefined dans la console

Le type : null

En Javascript, le type de données "null", indique une valeur qui est vide. 

let youtubeur = null; // On définis la variable à une valeur null.

Le type : Array (Tableaux)

Un tableau est une liste de valeurs de données entre crochets. Les tableaux peuvent contenir des types de données identiques ou différents. Les valeurs du tableau sont référencées par leur index. En JavaScript, l'index du tableau commence à zéro. C'est-à-dire que le premier élément d'un tableau se trouve à l'index zéro, le deuxième élément à l'index un et le troisième élément à l'index deux, etc.

["Chocolat", 59, "Nutella", false] // Création d'un Array qui contient plusieurs types de données.

L'opérateur typeof

L'opérateur typeof permet de savoir le type de données d'une valeur, variable etc.. (utile pour la compréhension du javascript)

console.log(typeof undefined); // Affichera "undefined"
console.log(typeof null); // Affichera "null"
console.log(typeof true); // Affichera "bool"
console.log(typeof 42); // Affichera "number
console.log(typeof Hello"); // Affichera "string"
console.log(typeof {}); // Affichera "object"
console.log(typeof []); // Affichera "object" (c'est un tableau)
