Les conversions de types en Javascript

En Javascript, il existe trois fonctions qui permettent de convertir un type en un autre :

  • Number() : C'est une fonction intégrée au Javascript, elle permet de transformer n'importe quelle valeur en un nombre.
  • String() : C'est une fonction intégrée au Javascript, elle permet de transformer n'importe quelle valeur en une chaine de caractères.
  • Boolean() : C'est une fonction intégrée au Javascript, elle permet de transformer n'importe quelle valeur en un booléen.
  • Conversions de chaînes de caractères en nombre :
    • parseInt() : Convertit une chaine de caractères en un nombre entier.
    • parseFloat() : Convertit une chaine de caractères en un nombre décimal.

La fonction Number()

La fonction Number() est utilisée en Javascript pour convertir n'importe quelle valeur en un nombre. Elle peut convertir n'importe quel type en sa représentation numérique.



Exemple de la conversion d'une chaîne de caractères (String) en un nombre (Number) :

Lorsque vous convertissez un mot ou une phrase comportant des symboles, des lettres, cela va afficher "Nan" (Not a Number).

let phrase = "Bonbons"; // Création d'une variable qui stocke un String
phrase = Number(phrase); // Convertis la variable "phrase" en un Nombre.
console.log(phrase); // Affiche : NaN (Not a Number)

==> Si vous convertissez un String qui contient un nombre comme : "56", elle va être immédiatement convertie en un nombre entier 56 qui va permettre les opérations.

Sans conversion

let nombrestr = "100"; // Création d'une variable qui stocke un nombre sous String
let nombre = 250; // Création d'une variable qui stocke un nombre entier
let resultat = nombrestr + nombre; // Création d'une variable qui stocke le résultat
console.log(resultat); // Affiche : 100250

Avec conversion

let nombrestr = "100"; // Création d'une variable qui stocke un nombre sous String
let nombre = 250; // Création d'une variable qui stocke un nombre entier
let resultat = Number(nombrestr) + nombre; // Création d'une variable qui stocke le résultat avec la conversion en nombre
console.log(resultat); // Affiche : 350


Exemple de la conversion d'un booléen (Boolean) en un nombre (Number) :

Lorsque vous convertissez un booléen en un nombre, true corresponderait au nombre 1 et false au nombre 0. 

console.log(Number(true)); // Console : 1
console.log(Number(false)); // Console : 0


Exemple de la conversion d'un null (pas de valeur) en un nombre (Number) :

Lorsque vous convertissez null en un nombre, une absence de valeur sera considérée comme le chiffre 0.

console.log(Number(null)); // Console : 0


Exemple de la conversion d'une valeur undefined (indéfini) en un nombre (Number) :

Lorsque vous convertissez undefined en un nombre, la valeur indéfini sera considérée comme le chiffre 0.

console.log(Number(undefined)); // Console : NaN

La fonction String()

La fonction String() est utilisée en JavaScript pour convertir n'importe quelle valeur en une chaîne de caractères. Elle peut convertir n'importe quel type en sa représentation sous forme de chaîne de caractères.



Exemple de la conversion d'un nombre (Number) en une chaîne de caractères (String) :

Lorsque vous convertissez un nombre en une chaîne de caractères, il sera représenté comme du texte.

let chaine = String(42); // Convertit le nombre en une chaîne de caractères
console.log(chaine); // Affiche : '42' (en tant que chaîne de caractères)


Exemple de la conversion d'un booléen (Boolean) en une chaîne de caractères (String) :

Lorsque vous convertissez un booléen en une chaîne de caractères, true sera représenté comme 'true' et false comme 'false'.

console.log(String(true)); // Affiche : 'true' (en tant que chaîne de caractères)
console.log(String(false)); // Affiche : 'false' (en tant que chaîne de caractères)


Exemple de la conversion d'une valeur null en une chaîne de caractères (String) :

Lorsque vous convertissez null en une chaîne de caractères, il sera représenté comme 'null'.

console.log(String(null)); // Affiche : 'null' (en tant que chaîne de caractères)


Exemple de la conversion d'une valeur undefined en une chaîne de caractères (String) :

Lorsque vous convertissez undefined en une chaîne de caractères, il sera représenté comme 'undefined'.

console.log(String(undefined)); // Affiche : 'undefined' (en tant que chaîne de caractères)

La fonction Boolean()

La fonction Boolean() est utilisée en JavaScript pour convertir n'importe quelle valeur en un booléen.



Exemple de la conversion d'une chaîne de caractères (String) en un booléen (Boolean) :

La conversion en booléen renvoie true pour toute chaîne de caractères non vide, sinon elle renvoie false.

console.log(Boolean(("Bonjour")); // Affiche : true
console.log(Boolean(" ")); // Affiche : false


Exemple de la conversion d'un nombre (Number) en un booléen (Boolean) :

La conversion en booléen renvoie false pour 0 et NaN, sinon elle renvoie true.

console.log(Boolean(0)); // Affiche : false
console.log(Boolean(42)); // Affiche : true


Exemple de la conversion d'une valeur null en un booléen (Boolean):

Lorsque vous convertissez null en un booléen, il sera représenté comme étant false.

console.log(Boolean(null)); // Affiche : false


Exemple de la conversion d'une valeur undefined en un booléen (Boolean) :

Lorsque vous convertissez undefined en un booléen, il sera représenté comme étant false.

console.log(Boolean(undefined)); // Affiche : false
