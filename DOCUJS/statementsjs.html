Les Instructions Conditionnelles en JavaScript

Les contrôles d'états (if / else / else if) permettent de prendre des décisions conditionnelles dans JavaScript. Ils permettent d'exécuter différentes parties de code en fonction de conditions spécifiques.

Voici les principaux mots-clés utilisés pour les contrôles d'états :

  • Utilisé pour exécuter un bloc de code si une condition est vraie.
  • Utilisé pour exécuter un bloc de code si la condition de l'if est fausse.
  • Utilisé pour spécifier une nouvelle condition si la première condition de l'if est fausse.
  • Utilisé pour exécuter un bloc de code en fonction de différentes conditions.
  • Permet d'évaluer une expression et d'exécuter différents blocs de code en fonction de la valeur de cette expression.

L'instruction "if"

Le mot-clé if est utilisé pour exécuter un bloc de code si une condition spécifiée est évaluée comme vraie (true). La condition peut être une expression qui est évaluée comme vraie ou fausse.

  • Syntaxe : 
    if(condition){
    bloc de code à effectuer
}
  • L'instruction if peut être suivie d'une instruction else optionnelle, qui permet d'exécuter un bloc de code si la condition spécifiée est fausse.
  • Si la condition fournie est vraie, le bloc de code à l'intérieur des accolades {} est exécuté. Sinon, si une instruction else est présente, le bloc de code dans else est exécuté.

L'instruction "else"

Le mot-clé else permet d'exécuter un bloc de code si la condition de l'if associé est fausse. Voici un exemple :

  • Syntaxe : 
    if(condition){
    bloc de code à effectuer
} else {
    bloc de code à effectuer
}
  • L'instruction else est utilisée conjointement avec l'instruction if pour exécuter un bloc de code alternatif si la condition de l'instruction if est évaluée à false. Cela permet de gérer les cas où la condition principale n'est pas remplie.

L'Opérateur Ternaire

L'opérateur ternaire, également appelé opérateur conditionnel, permet d'exécuter un bloc de code en fonction d'une condition donnée. Il s'agit d'un raccourci pour écrire des instructions conditionnelles.

  • Syntaxe : condition ? expressionSiVrai : expressionSiFaux;
  • L'opérateur ternaire se compose des éléments suivants :
    • condition : Une expression évaluée comme vraie ou fausse.
    • expressionSiVrai : L'expression ou le bloc de code exécuté si la condition est vraie.
    • expressionSiFaux : L'expression ou le bloc de code exécuté si la condition est fausse.

La fonction switch()

La fonction `switch()` permet d'évaluer une expression et d'exécuter différents blocs de code en fonction de la valeur de cette expression. Voici un exemple :

  • Syntaxe : 
    switch(expression) {
    case valeur1/condition1:
        // Bloc de code à exécuter si 
        // expression est égale à la 
        // valeur.
    break;
  • La fonction switch() se compose des éléments suivants :
    • case ... : : la valeur de la variable .
    • {...} : le bloc de code exécuté si la valeur correspond.
    • break; : L'expression pour sortir du code
    • On peut utilisé default : { bloc de code } à la fin du switch pour éxécuté un code si aucune case ne correspond à la valeur.
