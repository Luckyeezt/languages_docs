Les instructions conditionnelles en Javascript

En Javascript, il existe différentes façons afin d'éxécuter un bloc de code particulier en fonctions des conditions fournies :

  • if : C'est une instruction qui permet d'éxécuter d'un bloc de code si la condition fournie est vraie.
  • else : C'est une instruction de "secours" qui permet d'éxécuter un bloc de code différent si la condition de l'instruction if n'est pas vraie.
  • else if : C'est une instruction (comme le if) qui permet d'éxectuer un bloc de code si les conditions précédentes sont fausses.
  • L'opérateur ternaire : C'est un raccourci qui permet d'écrire une intruction conditionnelle (raccourci pour if/else)
  • La coalescence des nulls : C'est un opérateur introduit en JavaScript pour fournir une syntaxe concise pour sélectionner une valeur non nulle parmi plusieurs.
  • switch() : C'est une structure de contrôle en JavaScript utilisée pour effectuer des actions différentes en fonction de différentes conditions.

L'opérateur ternaire

L'opérateur ternaire, également appelé opérateur conditionnel, permet d'exécuter un bloc de code en fonction d'une condition donnée. Il s'agit d'un raccourci pour écrire des instructions conditionnelles.



Exemple sur la vérification d'un âge :

Ici, on va créer une condition qui va effectuer un bloc de code selon si la condition est respectée ou non.

let age = 17; // Création d'une variable qui stocke le nombre 17. 
let verif = age >= 18 ? console.log("Vous êtes majeur !") : console.log("Vous êtes mineur !")
  • L'instruction if peut être suivie d'une instruction else optionnelle, qui permet d'exécuter un bloc de code si la condition spécifiée est fausse.
  • Si la condition fournie est vraie, le bloc de code à l'intérieur des accolades {} est exécuté. Sinon, si une instruction else est présente, le bloc de code dans else est exécuté.
