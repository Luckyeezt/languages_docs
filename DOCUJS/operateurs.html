Les différents opérateurs arithmétiques en Javascript

  • L'opérateur d'addition (+) : C'est une séquence de caractères comme des lettres, chiffres, symboles pour représenter un texte en Javascript.
  • L'opérateur de soustraction (-) : Permet de représenter une valeur logique avec les booléens true(vrai) et false(faux).
  • L'opérateur de multiplication (*) : Permet de réprésenter des valeurs numériques comme un nombre entier ou un nombre décimal.
  • L'opérateur de division (/) : Permet de stocker une collection ordonnée d'élements, offre une manipulation de données efficace telle qu'une liste.
  • L'opérateur de puissance (**) : Permet de regrouper des données et des fonctionnalités dans une seule unité.
  • L'opérateur modulo (%) : Valeur par défaut si une variable à été déclarée mais aucune valeur lui à été assignée.

L'opérateur d'addition

L'opérateur d'addition (+) permet de combiner plusieurs valeurs entre elles, elles peuvent être des nombres, des chaînes de caractères, des variables contenant des valeurs etc...



Exemple d'addition de deux nombres :

Lorque l'on additionne seulement des nombres, le résultat attendu sera la somme des nombres additionnés. 

let calcul = 50 + 25 ; // Création d'une variable qui stocke une addition
console.log(calcul); // Affiche : 75


Exemple d'addition avec une chaîne de caractères et un nombre :

Lorsque l'on additionne des chaines de caractères avec des nombres on appelle ca une concaténation, le résultat attendu sera que le nombre va être ajouté à la chaîne de caractères.

let concat = "Chocolat" + 558 ; // Création d'une variable "concat" qui stocke une concaténation de strings et de nombres.
console.log(concat); // Affiche : Chocolat558.


Exemple d'addition avec deux chaines de caractères :

Lorsque l'on additionne des chaines de caractères, on appelle ca également une concaténation, le résultat attendu sera que les chaines de caractères vont se suivre en se collant.

let concat2 = "Chocolat" + "Croissant" + "Beurre"; // Création d'une variable "concat2" qui stocke une concaténation de strings.
console.log(concat2); // Affiche : ChocolatCroissantBeurre.

L'opérateur de soustraction (-)

L'opérateur de soustraction permet de soustraire une valeur à une autre.



Exemple de soustraction de deux nombres :

Lorsque l'on soustrait des nombres, le résultat attendu sera la différence entre les deux nombres.

let difference = 10000 - 3500; // Création d'une variable qui stocke le résultat de la soustraction
console.log(difference); // Affiche : 6500


Exemple de soustraction avec des variables :

On peut aussi soustraire des variables contenant des valeurs numériques.

let nombre1 = 100; // Création d'une variable "nombre1" qui stocke le nombre 100
let nombre2 = 75; // Création d'une variable "nombre2" qui stocke le nombre 75
let resultat = nombre1 - nombre2; // Création d'une variable qui stocke le résultat de la soustraction
console.log(resultat); // Console : 25

L'opérateur de multiplication (*)

L'opérateur de multiplication permet de multiplier deux nombres entre eux.



Exemple de multiplication de deux nombres :

Lorsque l'on multiplie des nombres, le résultat attendu sera le produit de la multiplication.

let produit = 10 * 5; // Création d'une variable qui stocke le résultat de la multiplication
console.log(produit); // Console : 50


Exemple de multiplication avec des variables :

On peut également multiplier des variables contenant des valeurs numériques.

let nombre1 = 20; // Création d'une variable "nombre1" qui stocke le nombre 20
let nombre2 = 3; // Création d'une variable "nombre2" qui stocke le nombre 3
let produit2 = nombre1 * nombre2; // Création d'une variable qui stocke le résultat de la multiplication (produit)
console.log(produit2); // Console : 60

L'opérateur de division (/)

L'opérateur de division permet de diviser un nombre par un autre.



Exemple de division de deux nombres :

Lorsque l'on divise des nombres, le résultat attendu sera le quotient de la division.

let quotient = 100 / 5; // Création d'une variable qui stocke le résultat de la division
console.log(quotient); // Console : 20


Exemple de division avec des variables :

On peut aussi diviser des variables contenant des valeurs numériques.

let nombre1 = 1000; // Création d'une variable "nombre1" qui stocke le nombre 1000
let nombre2 = 5; // Création d'une variable "nombre2" qui stocke le nombre 5
let quotient2 = nombre1 / nombre2; // Création d'une variable qui stocke le résultat de la division (quotient)
console.log(quotient2); // Console : 200

L'opérateur de puissance (**)

L'opérateur de puissance permet d'élever un nombre à la puissance d'un exposant.



Exemple de calcul de puissance :

Lorsque l'on utilise l'opérateur de puissance, le résultat attendu sera le nombre élevé à la puissance de l'exposant.

let resultatPuissance = 5 ** 3; // Création d'une variable qui stocke le résultat de la puissance
console.log(resultatPuissance); // Console : 125 (car 5^3 = 125)


Exemple de calcul de puissance avec des variables :

On peut également utiliser des variables pour effectuer des calculs de puissance.

let nombre1 = 2; // Définition d'une variable "nombre1"
let nombre2 = 4; // Définition d'une variable "nombre2"
let resultatPuissance2 = nombre1 ** nombre2; // Calcul de puissance avec des variables
console.log(resultatPuissance2); // Console : 16 (car 2^4 = 16)

L'opérateur de modulo (%)

L'opérateur de modulo (%) permet d'obtenir le reste de la division entre deux nombres.



Exemple de modulo :

En utilisant l'opérateur de modulo (%), on obtient le reste de la division entre deux nombres.

let resteDivision = 10 % 3; // Calcul du reste de la division
console.log(resteDivision); // Console : 1 (car 10 % 3 = 1)

Opérateurs d'Assignation Combinés

En JavaScript, les opérateurs d'assignation combinés permettent de raccourcir les expressions en combinant une opération arithmétique avec une opération d'assignation.



Voici les différents opérateurs d'assignation combinés :

  • += : Ajoute et assigne
  • -= : Soustrait et assigne
  • *= : Multiplie et assigne
  • /= : Divise et assigne
  • **= : Exponentiation et assigne
  • %= : Modulo et assigne

Exemples d'utilisation :

  let value = 10;
  
// Opérateurs d'assignation combinés
value += 5; // value devient 15
value -= 3; // value devient 12
value *= 2; // value devient 24
value /= 4; // value devient 6
value **= 2; // value devient 36
value %= 5; // value devient 1

value++ // Incrémentation (augmente de 1)
value-- // Décrémentation (diminue de 1)
