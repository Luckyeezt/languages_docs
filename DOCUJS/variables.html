Les variables en Javascript

En JavaScript, il existe trois façons de déclarer une variable, chacune peut stocker n'importe quel type de données :

  • const : C'est une variable à laquelle nous ne pourrons jamais changer sa valeur.
  • let : C'est une variable à laquelle nous pourrons changer sa valeur dans le futur.
  • var : Elle combine en quelque sortes les deux autres variables, elle peut changer sa valeur ou non dans le futur..

La création d'une variable :

  1. Ajout du mot clé de la variable :

    • const : Déclare une variable avec une valeur constante.
      • const nomDeVariable = valeur;
    • let : Déclare une variable avec une valeur modifiable.
      • let nomDeVariable = valeur;
    • var : Déclare une variable avec une portée globale ou de fonction.
      • var nomDeVariable = valeur;

  2. Le nom de la variable :

    • camelCase : Notation où la première lettre de chaque mot (sauf le premier) est en majuscule.
      • let produitVaisselle = valeur;
const ordinateurPortable;
    • snake_case : Notation où les mots sont séparés par des underscores "_".
      • const nom_de_variable = valeur;
    • Autres façons d'écrire un nom de variable :
      • let $profession = valeur;
let _reseau_social = valeur; 
let VIANDE = valeur;         
let JEU_VIDEO = valeur;
      • Règles :
      • - Un nom de variable JavaScript ne doit pas commencer par un chiffre.
      • - Un nom de variable Javascript ne doit pas contenir d'espaces
      • - Un nom de variable JavaScript n'autorise pas les caractères spéciaux à l'exception du signe dollar ($) et du trait de soulignement(_).
      • - Les noms de variables en Javascript sont sensibles à la casse.

    • La valeur de la variable :

      • données : Peut être un tableau, un nombre, un booléen, une chaîne de caractères, etc.
        • const firstProgram = "Hello World";
const secondProgram = 56;

Le déclaration de variable : const

Une variable déclarée avec "const" est utilisée pour stocker une valeur constante qui ne peut pas être réassignée une fois définie.

-> Les constantes sont souvent utilisés pour définir une valeur à laquelle nous sommes sûr qu'elles n'auront pas besoin d'une nouvelle valeur, ça améliore la lisiblité du code.

const boisson = "Le Coca-Cola";
const nombre_ventes = 150;
const marqueDeVoiture = "Mercedes";

console.log(boisson) // Console : Le Coca-Cola

Si jamais vous tentez de réassigner une variable constante, vous allez recevoir une erreur dans votre code, comme le montre l'exemple ci-dessous.

const telephone = "Iphone";
telephone = "Android";

console.log(telephone) // Console : TypeError: Assignment to constant variable.

La déclaration de variable : let

La déclaration de variable avec let permet de créer une variable dont la portée est limitée au bloc dans lequel elle est déclarée. Cette variable peut être réassignée avec de nouvelles valeurs au sein du même bloc.

Par exemple :

let prenom = "Rafael"; // Création d'une variable "prenom" qui stocke le prénom "Rafael"
prenom = "Nicolas"; // Réassigne la variable "prenom" au prénom "Nicolas".
          
console.log(prenom); // Affiche : Nicolas

Utilisez let lorsque vous avez besoin de réassigner des valeurs à une variable dans le même bloc de code. Si la valeur de la variable doit rester constante, utilisez plutôt const.

La déclaration de variable : var

La déclaration de variable avec var crée une variable dont la portée est limitée à la fonction ou au contexte global dans lequel elle est déclarée. Cette variable peut être réassignée avec de nouvelles valeurs à tout moment.

Par exemple :

var prenom = "Rafael"; // Création d'une variable "prenom" qui stocke le prénom "Rafael"
prenom = "Nicolas"; // Réassigne la variable "prenom" au prénom "Nicolas".
                    
console.log(prenom); // Affiche : Nicolas

Utilisez var lorsque vous avez besoin d'une variable dont la portée est limitée à la fonction ou au contexte global, et que vous souhaitez pouvoir la réassigner à tout moment. Cependant, il est recommandé d'utiliser let ou const pour des déclarations plus précises et sécurisées.


La déclaration var était un ancien moyen avec l'arrivée de let et const en 2015, var peut effectuer les deux déclarations const et let, juste il est déconseillé de l'utiliser à l'heure actuelle.


Contrairement au let, on peut réemployer la déclaration de variable lorsque l'on veut réassigner la variable à une nouvelle valeur : 

var nomDeCapitale = "Paris"; // Création d'une variable "nomdDCapitale" qui stocke le string "Paris"
var nomDeCapitale = "Tokyo"; // Réassigne la variable "nomDeCapitale" en stockant le string "Tokyo".
                      
console.log(nomDeCapitale); // Affiche : Tokyo
