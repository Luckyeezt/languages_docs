Les fonctions Math() en Javascript

Les fonctions mathématiques fournies par JavaScript via l'objet Math sont des outils puissants pour effectuer diverses opérations mathématiques en programmation. Elles permettent d'effectuer des calculs avancés et des manipulations numériques directement dans vos programmes JavaScript.


  • Math.PI : C'est une fonction qui permet d'accéder directement à la constante du nombre PI (π) qui est le rapport entre la circonférence d'un cercle et son diamètre.
  • Math.E : C'est une fonction qui accède à la constante mathématique "e" qui est la base du logarithme naturel.
  • Math.round() : C'est une fonction qui arrondit un nombre à l'entier le plus proche.
  • Math.floor() : C'est une fonction qui arrondit un nombre à l'entier inférieur le plus proche.
  • Math.ceil() : C'est une fonction qui arrondit un nombre à l'entier supérieur le plus proche.
  • Math.trunc() : C'est une fonction mathématique qui enlève la partie décimale d'un nombre.
  • Math.abs() : C'est une fonction qui retourne la valeur absolue d'un nombre.
  • Math.sign() : C'est une fonction qui renvoie 1 ou -1 selon si le nombre est négatif ou positif.
  • Math.pow() : C'est une fonction qui calcule la puissance d'un nombre.
  • Math.sqrt() : C'est une fonction qui calcule la racine carrée d'un nombre.
  • Math.sin() : Fonction qui calcule le sinus d'un nombre.
  • Math.cos() : Fonction qui calcule le cosinus d'un nombre.
  • Math.tan() : Fonction qui calcule la tangente d'un nombre.
  • Math.random() : C'est uen fonction qui retourne un nombre aléatoire entre x et y.
  • Math.min() : Fonction qui retourne le plus petit nombre d'une "liste" de nombres.
  • Math.max() : Fonction qui retourne le plus grand nombre d'une "liste" de nombres.

La fonction Math.PI

La fonction Math.PI est une méthode de l'objet Math qui renvoie la valeur de PI (π), une constante mathématique définie comme le rapport de la circonférence d'un cercle à son diamètre. Sa valeur est approximativement égale à 3.141592653589793.



La valeur de Math.PI

console.log(Math.PI); // Console : 3.141592653589793


Exemple de calcul de la circonférence d'un cercle 

const rayon = 5; // Création d'une variable "rayon" qui stocke le nombre 5.
const circonference = 2 * Math.PI * rayon; // Création de la variable qui va stocker la formule de la circonférence d'un cercle.
console.log(circonference); // Console : 31.4159263589793


Math.PI est souvent utilisé dans des formules impliquant l'utilisation du nombre PI comme par exemple :


  • Circonférence d'un cercle : 2 x Math.PI x rayon
  • Aire d'un cercle : Math.PI x rayon**2
  • Volume d'une sphère : (4/3) * Math.PI * Math.pow(rayon, 3)

La fonction Math.E

La fonction Math.E est une méthode de l'objet Math qui renvoie la valeur de "e" qui est la constante d'Euler, la base du logarithme naturel. Sa valeur est approximativement égale à 2.718281828459045.



La valeur de Math.E

console.log(Math.E); // Console :2.718281828459045


Exemple de calcul de l'exponentielle d'un nombre

L'exponentielle d'un nombre permet de calculer une croissance exponentielle.

// Arrondissement de nombres positifs : 
const nombre = 5; // Créer une variable "nombre" qui stocke le nombre 5.
const exponentielle = Math.E ** nombre; // Création de la variable qui va stocker la formule de calcul de l'exponentielle.
console.log("L'Exponentielle de " + nombre + " est " + exponentielle); // Console : L'Exponentielle de 5 est 148.4131591025766


Math.E vous permet d'accéder facilement à la valeur de la constante "e" pour effectuer divers calculs mathématiques impliquant des fonctions exponentielles et des logarithmes naturels.

La fonction Math.round()

La fonction Math.round() est une méthode de l'objet Math en JavaScript qui permet d'arrondir un nombre à l'entier le plus proche. Dans un nombre décimal, c'est à partir du x,5 que l'arrondissement change.



Exemple d'arrondissements de nombres avec Math.round() : 

// Arrondissement de nombres positifs : 
console.log(Math.round(1.49)) // Console : 1
console.log(Math.round(1.50)) // Console : 2
console.log(Math.round(1.51)) // Console : 2
                    
// Arrondissement de nombres négatifs : 
console.log(Math.round(-1.49)) // Console : -1
console.log(Math.round(-1.50)) // Console : -1
console.log(Math.round(-1.51)) // Console : -2


Exemple qui arrondis une température : 

let temperature = 26.40 // Création d'une variable "temperature" qui stocke le nombre décimal "26.40".
console.log(Math.round(temperature)) // Console : 26


  • Si la partie décimale est inférieure à 0.5, le nombre est arrondi vers le bas (vers l'entier le plus proche inférieur).
  • Si la partie décimale est supérieure ou égale à 0.5, le nombre est arrondi vers le haut (vers l'entier le plus proche supérieur).
  • Trad : "round" veut dire "rond" en francais, donc un cela représente un chiffre arrondi.

La fonction Math.floor()

La fonction Math.floor() est une méthode de l'objet Math en JavaScript qui retourne le plus grand entier inférieur ou égal à un nombre donné. Elle permet d'arrondir un nombre décimal vers le bas, en supprimant toute la partie décimal du nombre et en renvoyant l'entier le plus proche qui est inférieur ou égal au nombre d'origine.



Exemple d'arrondissements de nombres avec Math.floor() : 

// Arrondissement de nombres positifs : 
console.log(Math.floor(5.99)) // Console : 5
console.log(Math.floor(12.0)) // Console : 12
console.log(Math.floor(0.1)) // Console : 0
                    
// Arrondissement de nombres négatifs : 
console.log(Math.floor(-36.99)) // Console : -37
console.log(Math.floor(-58.0)) // Console : -58
console.log(Math.floor(-0.75)) // Console : -1


Exemple qui arrondis le prix d'un article : 

let prixArticle = 17.99; // Création d'une variable "prixArticle" qui stocke le nombre décimal "17.99".
let prixArrondi = Math.floor(prixArticle); // Créer une variable "prixArrondi" qui stocke la fonction Math.floor du prixArticle.
console.log("Prix arrondi : " + prixArrondi + " euros"); // Console : 17


  • Trad : "floor" veut dire "sol" en français, donc cela représente le nombre entier le plus bas.

La fonction Math.ceil

La fonction Math.ceil() est une méthode de l'objet Math en JavaScript qui retourne le plus petit entier supérieur ou égal à un nombre donné. Elle permet d'arrondir un nombre décimal vers le haut, en supprimant toute la partie décimale du nombre et en renvoyant l'entier le plus proche qui est supérieur ou égal au nombre d'origine.



Exemple d'arrondissements de nombres avec Math.ceil() : 

// Arrondissement de nombres positifs : 
console.log(Math.ceil(3.01)) // Console : 4
console.log(Math.ceil(12.5)) // Console : 13
console.log(Math.ceil(0.1)) // Console : 1
                        
// Arrondissement de nombres négatifs : 
console.log(Math.ceil(-3.01)) // Console : -3
console.log(Math.ceil(-12.5)) // Console : -12
console.log(Math.ceil(-0.1)) // Console : -0


Exemple qui cherche un pot plus grand pour sa plante : 

let taillePlante = 28.5; // Création d'une variable "taillePlante" qui stocke le nombre décimal "28.5".
let taillePot = Math.ceil(taillePlante); // Création d'une variable "taillePot" qui stocke la fonction Math.ceil sur la variable "taillePlante".
console.log("Hauteur de pot recommandé : " + taillePot + "cm"); // Console : Hauteur de pot recommandée : 29cm


  • Si le nombre contient une quelconque partie décimale, le nombre affiché sera immédiatement le nombre entier supérieur.
  • Trad : "ceil" est l'abbréviation de "CEILing" qui veut dire "plafond" en français, donc cela représente le nombre entier le plus haut.

La fonction Math.trunc()

La fonction Math.trunc() est une méthode de l'objet Math en JavaScript qui retourne la partie entière d'un nombre en supprimant la partie décimale. Elle permet de tronquer un nombre décimal pour ne conserver que sa partie entière, en ignorant les décimales.



Exemple de troncatures de nombres avec Math.trunc() : 

// Troncation de nombres positifs : 
console.log(Math.trunc(100.10)) // Console : 100
console.log(Math.trunc(0.1500)) // Console : 0
console.log(Math.trunc(50.75)) // Console : 50
                                            
// Troncation de nombres négatifs : 
console.log(Math.trunc(-100.10)) // Console : -100
console.log(Math.trunc(-0.1500)) // Console : -0
console.log(Math.trunc(-50.75)) // Console : -50


Exemple qui affiche un nombre tronqué : 

let number = 9.99; // Création d'une variable "number" qui stocke le nombre décimal "9.99".
let truncatedNumber = Math.trunc(number); // Création d'une variable qui effectue la troncature sur la variable "number".
console.log("La partie entière de " + number + " est " truncatedNumber); // Console : La partie entière de 9.99 est 9.


  • La fonction Math.trunc() supprime toute la partie décimale d'un nombre et renvoie la partie entière restante.
  • Trad : "trunc" veut dire "tronc" en français, on effectue une troncature sur un nombre.

La fonction Math.abs()

La fonction Math.abs() est une méthode de l'objet Math en JavaScript qui retourne la valeur absolue d'un nombre, c'est-à-dire sa valeur numérique sans tenir compte du signe. Elle permet de transformer un nombre négatif en positif en supprimant le signe négatif, tandis que les nombres positifs restent inchangés.



Exemples de l'utilisation de Math.abs() avec des nombres : 

// Utilisation de Math.abs() sur des nombres positifs : 
console.log(Math.abs(325)) // Console : 325
console.log(Math.abs(150)) // Console : 150
                                                
// Utilisation de Math.abs() sur des nombres négatifs : 
console.log(Math.abs(-675)) // Console : 675
console.log(Math.abs(-35)) // Console : 35


Exemple d'une distance entre deux points sur un axe numérique : 

let pointA = -10; // Création d'une variable "pointA" qui stocke le nombre négatif "-10".
let pointB = 15; // Création d'une variable "pointB" qui stocke le nombre positif "15".
let distance = Math.abs(pointA - pointB); // Création d'une variable qui met en valeur absolue le résultat du calcul pointA - pointB
console.log("La distance est de : " + distance) // Console : La distance est de : 25


  • Si le nombre passé en paramètre est négatif, la fonction Math.abs() transforme sa valeur en positif en supprimant le signe "-".
  • Si le nombre passé en paramètre est positif, la fonction Math.abs() retourne simplement le nombre inchangé.
  • Trad : "abs" est l'abbréviation de "absolute" qui veut dire "absolu(e)" en français, donc le nombre retourné est absolu.

La fonction Math.sign()

La fonction Math.sign() est une méthode de l'objet Math en JavaScript qui permet de déterminer le signe d'un nombre, en fonction s'il est positif ou négatif.



Exemples de l'utilisation de Math.sign() sur des nombres : 

console.log(Math.sign(50)); // Console : 1
console.log(Math.sign(-50)); // Console : -1
console.log(Math.sign(0)); // Console : 0
console.log(Math.sign("Chocolat")) // Console : NaN (Not a Number)


Exemple pour déterminer le signe d'un nombre :

let num = -10; // Création d'une variable "num" qui stocke le nombre négatif "-10".
let signValue = Math.sign(num); // Création d'une variable qui emploie la fonction Math.sign sur la variable "num".
console.log(signValue); // Console : -1


  • Math.sign renvoie 1 si le nombre est positif (+) ou 0 (zéro positif)
  • Math.sign renvoie -1 si le nombre est négatif (-) ou -0 (zéro négatif)
  • Math.sign renvoie "NaN" (Not a Number) si c'est autre chose que un nombre qui est passé en paramètres.
  • Trad : "sign" veut dire "signe" en français, donc c'est une fonction qui renvoie selon le signe d'un nombre.

La fonction Math.pow()

La fonction Math.pow() est une méthode de l'objet Math en JavaScript qui permet de calculer la puissance d'un nombre en utilisant une base et un exposant. Elle prend deux arguments : la base (nombre) et l'exposant (puissance), puis retourne la valeur de la base élevée à la puissance spécifiée.



Exemples de l'utilisation de Math.pow() sur des nombres : 

// Exemple de Math.pow() sur des nombres positifs :
console.log(Math.pow(3, 2)); // Console : 9 (3x3)
console.log(Math.pow(5, 3)); // Console : 125 (5x5x5)
console.log(Math.pow(10, 4)); // Console : 10 000 (10x10x10x10)


Exemple pour calculer l'aire d'un carré :

let longueurCote = 5; // Création d'une variable "longueurCote qui stocke le nombre "5".
const aireCarre = Math.pow(longueurCote , 2); // Création d'une variable qui emploie la fonction Math.pow sur la variable "longueurCote".
console.log(aireCarre); // Console : 25


  • La syntaxe de la fonction : Math.pow (base, exposant);
    • "base" : C'est le nombre dont vous souhaitez calculer avec une puissance.
    • "exposant" : C'est la puissance à laquelle ont veut calculer le nombre.
  • Trad : "pow" est l'abbréviation de "POWer" qui veut dire "puissance" en français. La fonction calcule effectivement la puissance d'une base à son exposant.

La fonction Math.sqrt()

La fonction Math.sqrt() est une méthode de l'objet Math en JavaScript qui permet de calculer la racine carrée d'un nombre. Elle prend un seul argument, qui est le nombre dont vous souhaitez calculer la racine carrée, et elle retourne la racine carrée de ce nombre



Exemples de l'utilisation de Math.sqrt() sur des nombres : 

// Exemple de l'utilisation de Math.sqrt() sur des nombres :
console.log(Math.sqrt(9)); // Console : 3
console.log(Math.sqrt(64)); // Console : 8
console.log(Math.sqrt(25)); // Console : 5

// Exemple de l'utilisation de Math.sqrt() sur un nombre négatif :
console.log(Math.sqrt(-50)) // Console : NaN (Not a Number)


Exemple d'un calcul de la racine carrée d'un nombre:

let num = 100; // Création d'une variable "num" qui stocke le nombre "100".
const racineCarree = Math.sqrt(num); // Création d'une variable qui emploie la fonction Math.sqrt sur la variable "num".
console.log(racineCarree); // Console : 10


  • Si le nombre passé en argument est négatif, la fonction Math.sqrt() retournera NaN (Not a Number) car la racine carrée d'un nombre négatif n'est pas un nombre réel dans le domaine des nombres réels.
  • Trad : "sqrt" est l'abbréviation de "SQuare RooT" qui veut dire "Racine Carrée" en français.

La fonction Math.sin()

La fonction Math.sin() est une méthode de l'objet Math en JavaScript qui permet de calculer le sinus d'un nombre. Elle prend un seul argument, qui est l'angle en radians dont vous souhaitez calculer le sinus, et retourne le sinus de cet angle.


  • Rappel Mathématique : Le sinus (sin) en mathématiques est une fonction trigonométrique qui associe à un angle dans un triangle rectangle le rapport entre la longueur du côté opposé à cet angle et l'hypoténuse du triangle.


Exemples de l'utilisation de Math.sin() sur des nombres : 

// Exemple de l'utilisation de Math.sin() sur des nombres positifs :
console.log(Math.sin(90)); // Console : 0.8939966636005579
console.log(Math.sin(180)); // Console : -0.8011526357338304

// Exemple de l'utilisation de Math.sin() sur un nombre négatif :
console.log(Math.sin(-360)); // Console : -0.9589157234143065


  • L'argument passé à Math.sin() est l'angle en radians pour lequel vous souhaitez calculer le sinus
  • Le sinus d'un angle dans un triangle rectangle est défini comme le rapport de la longueur du côté opposé à l'angle sur l'hypoténuse du triangle.
  • Trad : "sin" est l'abbréviation de "sine" qui veut dire "sinus" en français.

La fonction Math.cos()

La fonction Math.cos() est une méthode de l'objet Math en JavaScript qui permet de calculer le cosinus d'un angle en radians. Elle prend un seul argument, qui est l'angle en radians dont vous souhaitez calculer le cosinus, et retourne le cosinus de cet angle.


  • Rappel Mathématique : Le cosinus (cos) en mathématiques est une fonction trigonométrique qui associe à un angle dans un triangle rectangle le rapport entre la longueur du côté adjacent à cet angle et l'hypoténuse du triangle.


Exemples de l'utilisation de Math.cos() sur des nombres : 

// Exemple de l'utilisation de Math.cos() sur des nombres positifs :
console.log(Math.cos(90)); // Console : -0.4480736161291702
console.log(Math.cos(180)); // Console : -0.5984600690578581

// Exemple de l'utilisation de Math.cos() sur un nombre négatif :
console.log(Math.cos(-360)); // Console : -0.2836910914865273


  • L'argument passé à Math.cos() est l'angle en radians pour lequel vous souhaitez calculer le cosinus.
  • Le cosinus d'un angle dans un triangle rectangle est défini comme le rapport de la longueur du côté adjacent à l'angle sur l'hypoténuse du triangle.
  • Trad : "cos" est l'abbréviation de "cosine" qui veut dire "cosinus" en français.

La fonction Math.tan()

La fonction Math.tan() est une méthode de l'objet Math en JavaScript qui permet de calculer la tangente d'un angle en radians. Elle prend un seul argument, qui est l'angle en radians dont vous souhaitez calculer la tangente, et retourne la tangente de cet angle.


  • Rappel Mathématique : La tangente (tan) en mathématiques est une fonction trigonométrique qui associe à un angle dans un triangle rectangle le rapport entre la longueur du côté opposé à cet angle et la longueur du côté adjacent à cet angle.


Exemples de l'utilisation de Math.tan() sur des nombres : 

// Exemple de l'utilisation de Math.tan() sur des nombres positifs :
console.log(Math.tan(90)); // Console : -1.995200412208242
console.log(Math.tan(180)); // Console : 1.3386902103511544

// Exemple de l'utilisation de Math.tan() sur un nombre négatif :
console.log(Math.tan(-360)); // Console : 3.380140413960958


  • L'argument passé à Math.tan() est l'angle en radians pour lequel vous souhaitez calculer la tangente.
  • La tangente d'un angle est le rapport de la longueur du côté opposé à l'angle sur la longueur du côté adjacent dans un triangle rectangle.
  • Trad : "tan" est l'abbréviation de "tangent" qui veut dire "tangente" en français.

La fonction Math.random()

La fonction Math.random() en JavaScript est une fonction intégrée qui génère un nombre aléatoire compris dans l'intervalle [0, 1). Cela signifie que Math.random() retourne un nombre décimal pseudo-aléatoire qui peut être égal ou supérieur à 0 (inclus) et strictement inférieur à 1 (exclus).



Exemple qui génère un nombre aléatoire entre 0 et 10 :

L'utilisation de la fonction Math.floor permet de retirer la partie décimale en retournant le plus grand entier inférieur et génère ainsi un nombre aléatoire entier au lieu d'un nombre décimal.

let num = 10; // Création de la variable "num" qui stocke le nombre "10".
const gen = Math.floor(Math.random() * number); // Création de la variable "gen" qui emploie la fonction Math.random() sur la variable "num".
console.log(gen); // Console : Un nombre aléatoire entre 0 et 10.


Exemple qui génère un nombre entre 1 et 100 :

let randomNum = Math.floor(Math.random() * 100 + 1); // La plage de nombres aléatoires sera de 1 (inclus) à 101 (exclus), couvrant ainsi les nombres entiers de 1 à 100.
console.log(randomNum); // Console : Un nombre aléatoire entre 1 et 100.
  • La fonction Math.random() ne prend aucun argument.
  • Trad : "random" veut dire "aléatoire" en français.

La fonction Math.min()

La fonction Math.min() est une méthode de l'objet Math en JavaScript qui permet de trouver le plus petit nombre parmi un ensemble d'arguments ou d'éléments. Elle peut être utilisée pour retourner le nombre le plus petit dans une liste d'arguments ou dans un tableau.



Exemples qui cherche et affiche le nombre minimum : 

console.log(Math.min(5, 10, 2, 100)) // Console : 2


  • La fonction Math.min() peut prendre autant d'arguments numériques qu'elle veut.
  • Elle retourne le plus petit nombre parmi les valeurs fournies en argument.
  • Si vous fournissez à la fonction Math.min() des valeurs non numériques, elle tentera de les convertir en nombres avant de les comparer.
  • Trad : "min" est l'abbréviation de "minimum" en français.

La fonction Math.max()

La fonction Math.max() est une méthode de l'objet Math en JavaScript qui permet de trouver le plus grand nombre parmi un ensemble d'arguments ou d'éléments. Elle peut être utilisée pour retourner le nombre le plus grand dans une liste d'arguments ou dans un tableau.



Exemples qui cherche et affiche le nombre minimum : 

console.log(Math.max(5, 10, 2, 100)) // Console : 100


  • La fonction Math.max() peut prendre autant d'arguments numériques qu'elle veut.
  • Elle retourne le plus grand nombre parmi les valeurs fournies en argument.
  • Si vous fournissez à la fonction Math.max() des valeurs non numériques, elle tentera de les convertir en nombres avant de les comparer.
  • Trad : "max" est l'abbréviation de "maximmum" en français.
