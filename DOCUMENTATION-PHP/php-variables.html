Une variable est comme un "conteneur" qui va stocker une ou des données qui peuvent être modifiées au cours du programme et peut être re-exploitée plus tard dans un code.



Construction de variables

La syntaxe de la création d'une variable en PHP :

  1. Le signe "dollar" avant le nom de la variable : $
  2. Le nom de la variable : prenom
  3. Le signe "égal" après le nom de la variable afin de lui assigner une valeur: =
  4. La valeur de la variable (Types de données) : "Fréderic"
    5. $number1 = 25; // Création de la variable number1 qui stocker le nombre 25.
$email = "johndoe@gmail.com"; // Création de la variable email qui stocke un string

Une variables est correcte seulement si :

  • Elle commence avec le signe dollar ($).
  • Elle ne comporte pas d'espace dans son nom.
  • Elle commence par une lettre de l'alphabet (pas de symboles ni de chiffres)

Construction de variables constantes

Une variable constante contient une valeur qui restera inchangée durant tout le script du programme. Elle se créer avec la fonction define()

La syntaxe de la création d'une variable constante en PHP :

  1. define() -> Pour créer une constante il faut utiliser la fonction define()
  2. Le nom de la variable à l'intérieur des parenthèses : define("NOM_VAR")
  3. La valeur de la variable à l'intérieur des parenthèses : define("NOM_VAR", "VALEUR_VAR") 
    define("PRENOM", "Jonathan") // Création de la variable PRENOM qui stocke la valeur Jonathan.
define("NUMBER", 18) // Création de la variable NUMBER qui stocke le nombre 18.


Les manières d'afficher une variable constante

Imaginons que nous avons créer une variable et que nous voulons l'afficher, il y'a deux façons ! :

define("FOOD", "Hamburger") // Création de la variable FOOD qui stocke une nourriture.

Bien sûr nous devrons l'afficher avec la fonction écho : 

echo FOOD // Affichage de la variable FOOD.

Ou bien nous pouvons utiliser echo + la fonction constant()

echo constant("FOOD") // Affichage de la variable FOOD.
