La fonction var_dump()

La fonction var_dump() nous permet de connaître de quel type est la valeur d'une variable, cela peut être un nombre entier ou flottant, une chaine de caractères, un booléen etc...



Essayez de comprendre avec ces exemples ci-dessous : 

echo var_dump(25) // Console : int(25) -> Nombre entier
echo var_dump(3.15) // Console : float(3.15) -> Nombre entier
echo var_dump("Hello World!"); // Console : string(12) "Hello World!" -> String
echo var_dump(true); // Console : bool(true) -> Booléen

La fonction var_dump() nous renvoie plusieurs informations

  • Nombre entier : Type : int | Valeur entre parenthèses ()
  • Nombre flottant : Type : float | Valeur entre parenthèses ()
  • String : Type : string | Longueur de la valeur entre parenthèses () | Valeur " "
  • Booléen : Type : bool | Valeur entre parenthèses (true/false)
