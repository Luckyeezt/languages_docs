La fonction date() en PHP renvoie la date et l'heure actuelles en fonction de la configuration de votre serveur.



Fonction date() pour afficher la date

Paramètres pour la date :

  • l (L miniscule) : Représente le jour en lettres (Monday-Sunday)
  • d : Représente le jour du mois (0-31)
  • D : Représente les 3 premières lettres du jour de la semaine (Mon-Sun)
  • F : Représente le nom entier du mois (January - December)
  • m : Réprésente le mois sous format numérique (0-12)
  • M : Réprésente les 3 premières lettres du mois (Jan-Dec)
  • y : Représente l'année en deux chiffres
  • Y : Représente l'année en quatres chiffres

echo date("d/m/y") // Console : 11-07-24
echo date("D/M/Y") // Console : Thu-Jul-2024

Fonction date() pour afficher le temps

Paramètres pour le temps :

  • H : Format 24H d'une heure (00-23)
  • h : Format 12H d'une heure (01-12)
  • i : Représente les minutes (00-59)
  • s : Représente les secondes (00-59)

echo date("H:i:s") // Console : 03:46:50
