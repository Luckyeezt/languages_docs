Les "constantes magiques" sont des constantes déjà définies par le langage en lui-même, Elles sont très utiles pour obtenir des informations sur le fichier, la ligne, la fonction, la classe, etc., où elles sont utilisées.
En PHP il existe 9 "constantes magiques".


__LINE__

C'est une constante qui retourne le numéro de la ligne sur laquelle la variable est utilisée.

echo "Le numéro de la ligne est de : " . __LINE__ ;

Celà va produire dans la console :


Console : Le numéro de la ligne est de : 0



__FILE__

C'est une constante qui renvoie le chemin complet du fichier où la constante est utilisée

echo "Le chemin de ce fichier est : " . __FILE__ ;

Celà va produire dans la console :


Console : Le chemin de ce fichier est : C:\xampp\htdocs\website\learning.php



__DIR__

C'est une constante qui retourne le dossier du répertoire où le fichier utilisé se trouve.

echo "Le dossier du fichier est : " . __DIR__ ;

Celà va produire dans la console :


Console : Le dossier du fichier est : C:\xampp\htdocs\website



__FUNCTION__

C'est une constante qui retourne le nom de la fonction où se situe le script.

function hello() {
    echo "Ma fonction est : " . __FUNCTION__;
}
hello();

Celà va produire dans la console :


Console : Ma fonction est : test
