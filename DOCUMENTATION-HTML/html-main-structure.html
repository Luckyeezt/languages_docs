Structure de base - HTML

Pour commencer à créer une page html, il y'a 4 balises primordiales afin de créer une page en règle !


  • <!DOCTYPE html> : Cette balise est obligatoire pour toutes les pages HTML, elle permet de faire comprendre au navigateur quel type de fichier doit-il afficher, ici en l'occurence c'est du HTML.
  • <html> : L'élément html va contenir toute la page.
  • <head> : L'élément head va contenir les informations qui ne sont pas affichées sur la page, comme les métadonnées, les informations de style, les informations de script, les informations de références, etc...
  • <body> : L'élément body va contenir les informations qui sont affichées sur la page.

    • <!DOCTYPE html>
    <html>
        <head>

        </head>
        <body>

        </body>
    </html>
