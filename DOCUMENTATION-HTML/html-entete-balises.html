<title>

Permet de définir le titre de la page web (sur l'onglet).

<meta>

Permet de rajouter des informations sur la page web.


Attribut Valeurs Possibles Description
charset
  • UTF-8
 Spécifie le format de caractères d'une page.
name
  • Généralement author, description ou keywords
content
  • Texte
 Bref description de ce qui est indiqué dans l'attribut name.
