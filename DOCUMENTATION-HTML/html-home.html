C'est quoi le HTML

Le HTML dont les initiales signifient HyperText Markup Language est un langage de balisage qui utilise les chevrons ouvrants < & fermants > pour créer des balises, une balise contient une balise d'ouverture & de fermeture avec le slash (/), voici la syntaxe principale :

<nom_de_la_balise>.....</nom_de_la_balise>

Il existe également des balises qui n'ont pas de balises de fermetures, on les appellent des balises orphelines ou auto-fermantes, elles ne contiennent pas de textes.

<nom_de_la_balise/>
  • Crée en 1991 par Tim Berners-Lee
  • Définit et crée le contenu d'une page web
  • Gère le texte, vidéos, images, audios, liens vers d'autres pages etc..
